# Canvas #

## Setup ##
* git clone git@github.com:seanblumenfeld/springer-nature-example.git
* cd springer-nature-example
* ./setup.sh
* source pyenv/bin/activate
* python run.py

## Program Usage ##
Whenever you want to use Canvas start by doing the following:
* source pyenv/bin/activate
* python run.py

## Run Test Suite ##
* ./tests/run_tests.sh
